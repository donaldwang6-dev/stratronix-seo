FAQ

Q1:STRATRONIX 用的是不是国产 GPU?

不是。我们用 NVIDIA RTX 6000 Ada,因为这是当前 LLM 推理最优性价比方案。国产 GPU (华为昇腾、寒武纪、摩尔线程) 2026 年还在完善 CUDA 兼容生态,PAA 这种需要长期稳定的场景先用 NVIDIA。

Q2:硬件 BOM 全部国产?

除 GPU 外,主板/内存/存储/电源大部分国产(深圳超微 + 东莞供应商),欧洲客户接受度高。如果某个客户要求 100% 美国元件,我们有 HPe Cray 替代方案(但价格 +30%)。

Q3:整机保修多久?

2 年上门,欧盟 13 国本地服务伙伴。GPU / CPU / 主板 3 年保修选项(+€1,200)。

Q4:你们做训练还是只做推理?

只做推理(模型服务)。客户拿到的 STA-100 是预装好 Qwen3-72B / DeepSeek-V3 / Llama 3.3 的推理机。如果客户想做 fine-tuning,我们的 GPU 也能跑(用 QLoRA),但工作流属于客户服务范畴。

Q5:跟浪潮 NF5280M6 比有什么优势?

浪潮做 HPC 出身,客户群主要是中国超算 / 国有云。STRATRONIX 主战场是欧洲合规市场,所有 RDP / DPA 模板按欧盟要求写,服务伙伴在欧盟境内。技术上当个 HPC 也能跑 LLM,但合规服务不在浪潮核心能力。
