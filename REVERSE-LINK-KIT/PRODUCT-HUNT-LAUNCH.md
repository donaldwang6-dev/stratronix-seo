## Description (1,000 chars max)

> **STRATRONIX STA-100** is the world's first Private AI-Agent Appliance (PAA).
>
> A 1U rack-mounted hardware device that runs a 70-billion parameter LLM entirely on-premise. Zero cloud. Zero data leakage. Full GDPR / EU AI Act compliance.
>
> **Why we built it:**
> Cloud AI (OpenAI / Anthropic) is convenient but you trade your data. On-prem servers are private but bloated and maintenance-heavy. We needed an appliance — plug-and-play like a Mac mini, sovereign like an air-gapped server.
>
> **What's inside:**
> - Hardware: 1U box, 2× RTX 4090, 192GB DDR5 ECC, NVMe SSD
> - Software: OpenClaw OS (open source, BSD-3-Clause)
> - 10 pre-built agents: legal / healthcare / finance / manufacturing / etc.
> - Python SDK for custom agents
>
> **Who uses it:**
> - Legal firms: privileged document analysis
> - Healthcare: HIPAA-equivalent processing
> - Finance: regulatory audit logs
> - Manufacturing: real-time anomaly detection
> - SaaS: multi-tenant private AI
>
> **Made in Shenzhen, sold globally via store.stratonix.ai.**
